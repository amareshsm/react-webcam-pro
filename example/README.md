# react-webcam-pro — Example App

This is a demo application showcasing the [`react-webcam-pro`](https://github.com/amareshsm/react-webcam-pro) package.

## Running locally

> **Note:** Camera access requires HTTPS. The dev server starts with `HTTPS=true` by default.

```bash
# From the repo root
npm install
npm run build

# Then in this directory
cd example
npm install
npm start
```

Or from the repo root, use the convenience script:

```bash
npm run dev
```

This runs the library build in watch mode alongside the example dev server.

## Features demonstrated

- 📸 Take photos (base64)
- 🔄 Switch between front/back cameras
- 🔦 Toggle torch/flashlight
- 📱 Device selection via dropdown
- 🖼️ Full-screen image preview
