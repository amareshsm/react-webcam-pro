Community & Support

react-webcam-pro is a community-maintained fork of react-camera-pro. We're committed to keeping this package actively maintained and responsive to the community's needs.

🔧 Open Issues​

We're steadily working through the open issues inherited from the original react-camera-pro repository. Many of the long-standing bugs — including React 19 compatibility, styled-components v6 warnings, and camera switching problems — have already been fixed in v1.0.

Remaining issues are being triaged and fixed on a rolling basis.

🚨 Need Something Fixed Urgently?​

If you encounter a bug or need a feature that's blocking your work, please create an issue in our repository. Issues reported directly to us will be taken up on high priority and addressed quickly.

How to Report​

When creating an issue, please include:

  • Description — What happened vs. what you expected
  • Steps to reproduce — Minimal code or repo link
  • Environment — React version, styled-components version, browser, OS
  • Screenshots / logs — If applicable
